Site Administration

The intention of this application is for administrator to configure this intranet site. They manage user lists, the groups they belongs to, and the security rights they have to each application. All users for this site other than the guest user will be entered into the system via this application. The items on the left side bar and its submenu are also managed by this application. Due to security concerns, this application required a sign on by a user with administrator rights. This is either an administrator or user belonging to the administration group. All sub menus within this application check security rights frequently to prevent a security breach.
